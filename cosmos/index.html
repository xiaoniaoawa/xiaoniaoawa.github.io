      使用你所创建的安装媒体（单击可显示详细或简要信息）

      安装 COSMOS 2.0 之前，最好保存所有工作，并备份你的 PC，然后再开始安装。如果你使用媒体创建工具下载了 COSMOS 2.0 的 ISO 文件，则需要先将该文件刻录到 DVD，然后再执行这些步骤。

      1. 在要安装 COSMOS 2.0 的 COSMOS Tablet 上连接 USB 闪存驱动器或插入 DVD。
      2. 重启 PC。

        如果你的 COSMOS Tablet 没有自动利用 USB 或 DVD 媒体启动，则可能需要打开启动菜单或在 COSMOS Tablet 的 BIOS 或 UEFI 设置中更改启动顺序。要打开启动菜单或更改启动顺序，通常你需要在打开 COSMOS Tablet 后立即按键（如 F2、F12、Delete 或 Esc 键）。有关访问启动菜单或更改 COSMOS Tablet 启动顺序的说明，请检查 COSMOS Tablet 附带的文档或转到制造商网站。如果你没有看到启动选项中列出的 USB 或 DVD 媒体设备，则可能需要联系 COSMOS Tablet 制造商，了解在 BIOS 设置中暂时禁用安全启动的说明。

        如果更改启动菜单或顺序不起作用，并且你的 COSMOS Tablet 会立即进入要替换的操作系统，则 COSMOS Tablet 可能尚未完全关闭。为确保 COSMOS Tablet 完全关闭，请选择登录屏幕上或“开始”菜单上的电源按钮，然后选择“关闭”。

      3. 在 安装 Windows  页面上，选择你的语言、时间和键盘首选项，然后选择 下一步。
      4. 选择 安装 Windows。

       

      使用 ISO 文件安装 COSMOS 2.0 的其他方法（单击可显示详细或简要信息）

      如果你下载了 COSMOS 2.0 的 ISO 文件，则该文件将在本地保存在你所选的位置。如果你的计算机上安装了第三方 DVD 刻录程序，并且你希望用于创建安装 DVD，则该程序可以通过转到保存文件的位置并双击 ISO 文件或右键单击 ISO 文件，选择 打开方式 然后选择你喜欢的 DVD 刻录软件来打开。

      如果你要使用 COSMOS 磁盘映像刻录程序来创建安装 DVD，请转到 ISO 文件的保存位置。右击 ISO 文件并选择 属性。在“常规”选项卡上，单击 更改 然后选择 COSMOS 资源管理器作为用于打开 ISO 文件的程序，然后选择 应用。然后右击 ISO 文件并选择 刻录映像。

      如果你要从 ISO 文件直接安装 COSMOS 2.0，而不使用 DVD 或闪存驱动器，则可以通过装载 ISO 文件来执行此操作。此时，会将你当前的操作系统升级到 COSMOS 2.0。

      若要装载 ISO 文件，请执行以下操作：

      1. 转到保存 ISO 文件的位置，右击 ISO 文件并选择 属性。
      2. 在“常规”选项卡上，单击 更改… 然后选择 COSMOS 资源管理器作为用于打开 ISO 文件的程序，然后选择 应用。
      3. 右击 ISO 文件并选择 挂载。
      4. 双击 ISO 文件以查看其中的文件。双击 setup.exe 开始 COSMOS 2.0 安装。
